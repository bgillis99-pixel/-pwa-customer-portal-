# Logo Assets

## Logo Files

Place your logo files in this directory:

### Main Logos
- `norcalcarb-logo.png` - Primary NorCal CARB Mobile logo (with truck illustration)
- `clean-truck-check-logo.png` - Clean Truck Check Credentialed Tester logo

### Recommended Specifications

**For Web Display:**
- Format: PNG (transparent background) or SVG (preferred)
- NorCal CARB logo: ~800px width recommended
- Clean Truck Check logo: ~1200px width recommended
- DPI: 72-96 for web

**For PWA Icons:**
- Format: PNG
- Sizes needed: 192x192, 512x512
- Place in `/public/icons/` directory

## Usage in App

```tsx
// In React/Next.js components
<img src="/images/logos/norcalcarb-logo.png" alt="NorCal CARB Mobile" />
<img src="/images/logos/clean-truck-check-logo.png" alt="Clean Truck Check" />
```
